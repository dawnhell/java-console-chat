Приложение будет состоять из следующих классов:
1. Server
2. Client
3. ClientGUI
4. Message

Класс Server: будет реализовывать все базовые методы, такие как запуск, остановка сервера, отображение сообщений пользователей, добавление нового пользователя, удаленеи пользователя по его id.
Класс Client: будет реализовывать: запуск и завершение клиентского приложения, подключаться/отключаться к/от серверу, отправлять сообщения.
Класс ClientGUI: будет реализовывать графическую оболочку для Client. Пока не знаю, как и с помощью чего он будет это делать.
Класс Message - определяет различные типы сообщений, которыми мы будем обмениваться. Будем отправлять  Java объекты т.к. это намного удобнее, чем использовать обычные строки(нам не надо будет считать все байты или ждать окончания строки).
